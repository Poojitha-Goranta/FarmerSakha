FarmerSakha is a web-based crop advisory system that analyzes soil and climate parameters such as pH, moisture, temperature, light intensity, and nutrient levels to recommend suitable crops and provide yield improvement strategies.

## Features
- Soil health analysis
- Crop suitability scoring
- Yield improvement advice
- Cost-effective soil remedies
- Cloud-based data integration using Google Sheets

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Python (Flask)  
Data Source: Google Sheets (CSV), Agricultural Dataset  

## How It Works
1. Soil data is entered into Google Sheets.
2. Backend reads the latest input values.
3. System compares input with crop dataset.
4. Recommendations are displayed on the website.

## Future Improvements
- Real-time IoT sensor integration
- Region-specific datasets
- Machine learning-based prediction
